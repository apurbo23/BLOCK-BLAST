Block Blast

A simple Block Blast puzzle game developed in Java as a university software development project.

Description:

The objective of this project is to recreate the popular Block Blast puzzle game and gain practical experience in game development concepts such as object-oriented programming, event handling, game logic implementation, score management, and graphical user interface design.
The player is required to place different shaped blocks on a grid strategically. When a complete row or column is filled, it is cleared from the board and the player earns points. The game continues until there is no available space left to place the remaining blocks.
The project will be developed step by step, beginning with the core gameplay mechanics and gradually adding improvements and additional features to enhance the user experience.

Features:

Drag and place block pieces onto the game board.
Multiple block shapes and sizes.
Clear completed rows and columns automatically.
Real-time score tracking system.
High score saving functionality.
Game over detection.
Restart game option.
User-friendly graphical interface.
Smooth gameplay experience.
Sound effects and animations (optional).

Technologies:

Java
Java Swing / Java AWT
Object-Oriented Programming (OOP)
